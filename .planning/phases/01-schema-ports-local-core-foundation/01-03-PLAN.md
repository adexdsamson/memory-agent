EmbeddingProvider Protocol (from ports/embedding.py, implemented in Plan 02):
  class EmbeddingProvider(Protocol):
      @property
      def dim(self) -> int: ...
      async def embed(self, texts: list[str]) -> list[list[float]]: ...
  # Contract: embed() ALWAYS returns L2-normalized (unit) vectors

StubEmbedder implementation pattern (from RESEARCH.md Pattern 6):
  - SHA-256 hash of text → 32 bytes → build dim-length vector by cycling digest bytes
  - raw[i] = digest[i % 32] / 255.0   for i in range(dim)
  - L2-normalize: norm = sqrt(sum(x*x for x in raw)) or 1.0; result = [x/norm for x in raw]
  - dim=128 is the standard test size; must be configurable via __init__(dim=128)
  - No numpy dependency in stub (use stdlib hashlib + math) — keeps it a zero-dependency helper

Scheduler Protocol (from ports/scheduler.py, implemented in Plan 02):
  class Scheduler(Protocol):
      def schedule(self, fn: object, *, every_seconds: int) -> None: ...
      def trigger_now(self) -> None: ...
      def start(self) -> None: ...
      def shutdown(self) -> None: ...

InProcessScheduler implementation pattern (from RESEARCH.md Pattern 4 — VERIFIED):
  from apscheduler.schedulers.asyncio import AsyncIOScheduler
  from datetime import datetime

  JOB_ID = "consolidate"
  _scheduler = AsyncIOScheduler()

  def schedule(self, fn, *, every_seconds):
      self._job = self._scheduler.add_job(fn, "interval", seconds=every_seconds,
                                          id=JOB_ID, next_run_time=None)

  def trigger_now(self):
      job = self._scheduler.get_job(JOB_ID)
      if job:
          job.modify(next_run_time=datetime.now())   # fires within ~100ms

  def start(self):   self._scheduler.start()
  def shutdown(self): self._scheduler.shutdown(wait=False)

  CRITICAL: Use "apscheduler>=3.11,<4" — 4.x is alpha; AsyncIOScheduler is in the 3.x API
  CRITICAL: next_run_time=None on add_job means "don't fire immediately on first schedule"
  CRITICAL: trigger_now() uses datetime.now() (local time), not datetime.utcnow()
