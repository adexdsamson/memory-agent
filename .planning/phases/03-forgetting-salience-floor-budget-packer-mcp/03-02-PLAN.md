    re_rank(records, similarity_scores, now=None) -> list[MemoryRecord]:
    - composite(r) = similarity_scores.get(r.id, 0.5) * r.salience * exp(-LAMBDA_DECAY * age_days)
    - age_days from last_accessed if set, else created_at (same reference as keep_score)
    - Returns records sorted descending by composite score

    TokenCounter Protocol:
    - count(self, text: str) -> int — synchronous (D-12)
    - No @runtime_checkable (D-10)

    TiktokenCounter:
    - Uses tiktoken.get_encoding("cl100k_base") — import deferred into __init__ (noqa: PLC0415)
    - count() returns len(enc.encode(text))

    ByteLengthCounter:
    - count() returns max(1, len(text.encode("utf-8")) // 4)
    - Zero runtime dependencies — portable fallback

    pack_records(ranked, budget, counter) -> list[MemoryRecord]:
    - Pass 1: collect CRITICAL_SET = {r for r in ranked if r.protected or (r.record_type == RecordType.FACT and r.valid_until is None)}
    - Fit critical records under budget (protected first, then FACT — use list order from ranked to preserve score ordering)
    - Pass 2: fill remaining budget from ranked excluding already-packed IDs
    - Use continue (NOT break) for oversized records in Pass 2 (RESEARCH.md Pitfall 3)
    - Budget unit: tokens counted by counter.count(r.summary or r.content[:80])
  </behavior>
