| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-05 | Tampering | supersede() — cross-user record retirement | mitigate | `AND user_id=?` predicate in the UPDATE statement. Consolidation pipeline always passes the correct user_id from the staging queue item (D-02/D-03). |
| T-02-06 | Tampering | supersede() — partial write on exception | mitigate | try/except + rollback pattern; commit only on full success. aiosqlite single-connection WAL ensures rollback is atomic. |
| T-02-07 | Tampering | find_by_t0_ref() — returns superseded provisional | mitigate | `AND valid_until IS NULL` predicate excludes superseded records from idempotency lookup. |
