  <behavior>
    keep_score(record, now=None) → float:
    - Uses last_accessed if set, else created_at as the reference time (D2-15)
    - age_days = max(0.0, (now - ref_time).total_seconds() / 86400.0)
    - recency = math.exp(-LAMBDA_DECAY * age_days)  where LAMBDA_DECAY = 0.05
    - reinforce = math.log(1.0 + float(record.access_count))
    - score = W_RECENCY * recency + W_REINFORCE * reinforce + W_SALIENCE * record.salience
    - Returns min(1.0, max(0.0, score))
    - W_RECENCY=0.4, W_REINFORCE=0.3, W_SALIENCE=0.3 (tunable module-level constants)
    - D-12: no I/O, no async, no imports at runtime beyond math + datetime

    decay_pass(record_store, user_id, now=None) — async generator:
    - Iterates record_store.live_records(user_id)
    - For each record: if record.protected → yield (record, 1.0) — protected are never scored below 1.0
    - For non-protected: yield (record, keep_score(record, now))
    - NOTE: the FORG-03 invariant (skip protected BEFORE score math) is implemented here by
      yielding protected records with score=1.0 — the eviction caller in Phase 3 treats any
      score >= floor as "keep" so protected records always survive. The test_protected_skipped_before_score_math
      test verifies this by calling keep_score directly on a protected record (it must not raise,
      confirming the guard is in the caller, not inside keep_score)

    Test cases (RED → GREEN):
    - test_keep_score_values: construct MemoryRecord with known fields; call keep_score with a
      fixed `now` datetime; assert float result matches formula to within 1e-6 tolerance;
      assert 0.0 <= score <= 1.0
    - test_protected_skipped_before_score_math: call keep_score directly on a protected=True record;
      it must return a valid float (keep_score is not the guard) AND verify that in decay_pass the
      protected record yields score=1.0 regardless of age
  </behavior>

    def keep_score(record: "MemoryRecord", now: datetime | None = None) -> float:
      Docstring: "Pure sync keep_score per D-12 — no I/O, no async. Reference time:
      last_accessed if set, else created_at (D2-15). Protected-record skip is the CALLER's
      responsibility (FORG-03); keep_score computes the score regardless of protected flag."
      Formula as specified in behavior above.

    async def decay_pass(
        record_store: "Any",
        user_id: str,
        now: datetime | None = None,
    ) -> AsyncIterator[tuple["MemoryRecord", float]]:
      Docstring: "Async generator yielding (record, keep_score) for all live records.
      Protected records yield score=1.0 (FORG-03 structural guarantee in caller path)."
      if now is None: now = datetime.now(timezone.utc)
      async for record in record_store.live_records(user_id):
          if record.protected:
              yield (record, 1.0)
          else:
              yield (record, keep_score(record, now))
